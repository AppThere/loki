# 0001. ISO/IEC 29500-2:2021 as Primary Authority

## Context
We need to decide which version of the Open Packaging Conventions (OPC) specification to implement.

## Decision
ISO/IEC 29500-2:2021 (Fourth Edition) is the primary authority.

## Rationale
ECMA-376 (fifth edition, Part 2) is technically equivalent for OPC, but ISO is the more carefully maintained version for our purposes. The 2021 edition corrected the empty media-type ambiguity and clarified pack IRI handling and non-ASCII part name encoding — all issues that routinely affect real-world files. Using the latest edition means we implement the most mathematically correct version of the spec.

## Consequences
Files manually generated by `loki-opc` natively conform to ISO 29500-2:2021 Strict guidelines. Older legacy OOXML implementations failing conformity checks map to the specific compatibility shims logically correcting inputs into valid streams locally.
